## ℹ️ What is mcp-accessibility-bridge?

mcp-accessibility-bridge is a server application. It helps you get Chrome's accessibility information. This is useful if you want to test websites using natural language. For example, mcp-accessibility-bridge allows you to turn simple questions into selectors for testing tools like Playwright, Selenium, Cypress, and WebdriverIO.

This means you do not need to know how to write complex code to find elements on a web page. The server reads the accessibility tree from Chrome and connects that to Claude, a language model. Together, they help generate selectors based on what you describe.

## 🔄 How It Works

- You run the server on your Windows machine.
- The server connects with Chrome through the Chrome DevTools Protocol.
- When you ask questions or type instructions in natural English, the server uses Claude, an AI model.
- Claude understands your request and finds the right selectors for testing.
- This works with popular testing frameworks like Playwright, Selenium, Cypress, and WebdriverIO.
